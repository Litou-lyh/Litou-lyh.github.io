2. Floating points

2.1 IEEE Standard for fp

S EXP (shift) / E Frac (value) / M
1 bit 8 bits 23 bits
(+ / -) $\times 2 ^ E, E = Exp - 127$, shifting the fp
$E = 1 - bias$ if Exp == 0		 1.xxxxxx, first 1 is hidden

Note: if Exp == 00…0 && Frac == 0 , fp == 0 (since we assume 1.xx start, need a special value)

​ if Exp == 00…0 && Frac != 0, denormalized numbers (small numbers)

​ if Exp == 11…1, fp == $\infty$ (NaN)

2.2 Floating point rounding

Round to nearest even: consider statistical effect (0 is the center!)

7.85 -> 7.8; 7.95 ->8.0

2.3 Floating point Multiplication

if $E$ overflow, fp overflow
